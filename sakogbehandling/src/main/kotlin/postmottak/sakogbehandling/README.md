# Sak og behandling
Fremtidig modul for sak og behandling inkludert ident.

Modulen vil ha ansvar for å håndtere relasjonen mellom en sak og en person (les ident),
og relasjonen mellom en sak og én eller flere behandlinger.

Sak og behandling vil være relativt enkle modeller, og vil først og fremst være viktige
å svare ut behov knyttet til sporbarhet ifm. forvaltningsloven.

SakId og spesielt behandlingId vil anvendes som nøkkel inn mot andre subdomener for å knytte
faktagrunnlag, vilkår og avklaringsbehov til saken og behandlingen.
